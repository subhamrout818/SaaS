

# Team Workflow

## Before Starting Work

- Pull the latest changes.
- Read the assigned scene.
- Read the design system if needed.

---

## Before Pushing

Do NOT push directly.

Post in the WhatsApp group:

- What changed
- Which files changed
- Screenshots (if UI changed)

Wait for approval.

Then push.

---

## Coding Rules

- Follow the Design System.
- Follow the Creative Bible.
- Keep commits focused.
- Never push broken code.
